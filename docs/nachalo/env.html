Env

.Env

Файл содержит переменые окружения. Когда фреймворк запускает, все данные этого файла попадают в супер-глобальный массив $_ENV

  • APP_ENV - Название среды разработки, все конфиги ниже относятся к этой среде разработки.
  • APP_KEY - Ключ шифврования
  • SESSION_DRIVER - Настрйока сессий
  • CACHE_DRIVER - Настройка кэша
  • QUEUE_DRIVER - Настройка очередей
